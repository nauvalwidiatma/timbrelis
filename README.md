# Timbrelis GitHub Pages Privacy Site

Upload the contents of this `privacy-site` folder to a GitHub repository configured for GitHub Pages.

Recommended public URL:

`https://YOUR_GITHUB_USERNAME.github.io/timbrelis/privacy/`

Replace `YOUR_GITHUB_USERNAME` with the actual GitHub username or organization.

Google Play requires the Privacy Policy URL to be active, public, non-geofenced, non-editable by visitors, and not a PDF.
